# User Profile & Preferences

**Last Updated:** 2026-01-17

## Working Style

### Decision Making
- **Autonomy Preferred**: User prefers agents make decisions independently without asking for approval
- **Order of Execution**: Agent should decide implementation order and approach
- **Communication**: User doesn't want to know step-by-step details unless critical issues arise

### Values & Priorities
- **Continuous Learning**: Highly values systems that learn and improve over time
- **Knowledge Sharing**: Wants learnings to persist and be accessible across all agent sessions
- **Forward-Thinking**: Interested in meta-improvements to the system itself
- **Efficiency**: Prefers agents work autonomously rather than seeking constant validation

## Character Traits
- **Trust**: Demonstrates high trust in AI agents by giving full autonomy
- **Systems Thinker**: Thinks about infrastructure and foundational systems before features
- **Learning-Oriented**: Values knowledge accumulation and sharing mechanisms
- **Pragmatic**: Wants real work done, not just planning and discussion

## Technical Preferences
- **Persistence**: Knowledge should survive across sessions
- **Accessibility**: All agents should access shared knowledge base
- **Structured Knowledge**: Prefers organized, categorized information

## Communication Preferences
- **Results-Focused**: Show outcomes, not process
- **Autonomous Execution**: Make decisions and execute without asking
- **Meta-Awareness**: Appreciates when agents improve themselves

---

**Note to Agents**: Before asking the user for decisions, check if you can make a reasonable choice autonomously. The user trusts your judgment.
