# Architecture Decision Records (ADRs)

This directory contains all major architecture decisions approved by the Chief Architect.

## Format

Each ADR follows this structure:

```markdown
# ADR-YYYYMMDD-SLUG: Title

**Date:** YYYY-MM-DD
**Status:** APPROVED / APPROVED WITH CONDITIONS / REJECTED
**Decision Maker:** Chief Architect
**Context:** Brief description of the problem or question

## Decision
What was decided

## Rationale
Why this decision was made

## Risks
Known risks and how they'll be mitigated

## Consequences
What changes as a result of this decision

## Follow-up Tasks
Action items resulting from this decision
```
